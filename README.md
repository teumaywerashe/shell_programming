# USSD Message Simulator

A lightweight **Shell Script** designed to simulate the user experience of a Telecommunications USSD (Unstructured Supplementary Service Data) menu. This program demonstrates basic command-line interaction, input handling, and conditional logic in a Unix-like environment.

## 🚀 Features
*   **Interactive Menu:** Clean terminal interface for navigating service options.
*   **Input Validation:** Handles user choices and provides feedback for invalid entries.
*   **Modular Logic:** Built using `case` statements for easy expansion of menu items.
*   **Simulated Backend:** Demonstrates how "Balance Checks," "Data Top-ups," and "Airtime Transfers" would flow.
